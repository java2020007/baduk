시작하기

  난이도: 입문자   키워드: SL 설명, 색인 페이지

여기는 Sensei's Library에서 바둑학습을 시작할 수 있는 출발점입니다. (바둑학습의 시작 지점을 찾는 경우, 첫번째 이동을 참조하세요.)

메인 메뉴

  • 초보자를 위한 페이지 - 만약 바둑을 처음 둔다면, 여기서부터 시작하세요.
  • Reference Section - a more encyclopaedic approach with top-down structure.
  • Guided Tours - follow themed tours through the library - recommended!
  • 페이지 찾기 - title search, full text search, keyword search
  • Find Board Positions - full-featured board search over all SL content (partial or whole board)
  • Recent Changes - pages that are currently watched and modified.
  • Most Popular - the most popular pages on SL (see also: SL Statistics)
  • Random Page - go to a random page in the library

Contributing to the library

  • Sensei's Library - our mission statements
  • How to use Wiki and Understanding Wiki Pages - how you can participate in the Sensei's Library project, technical help
  • Library Work an outline of some key areas that need to be worked on
  • User Preferences - set your name and make other customizations
  • Meta Discussion - feedback on the content of Sensei's Library (SL)
  • Bug Report - feedback on the technology behind SL and its usage

Quick stops

  • Go News - latest events in the world of Go
  • Message Board - messages which are only relevant for a short span of time
  • Quick Questions - quick questions!
  • Coffee Machine - motto of the day and other gossip
  • Tour Bus Stop - take a tour to other wikis
  • Link Collection - links to other web sites

Other noteworthy pages

  • Wiki News - announcements from the administrators
  • Wiki Indexes - a different view
  • HomePages - people's own SL pages
  • Pages by Distance and Distance from Frontpage - how well pages are linked
  • Touchgraph Wiki Browser - visualizing the link structure of this wiki
  • SL Snapshot - if you need a local copy of SL
  • SL Journal - pedagogical, editorial, and other material
